Character Recognition System

Using image processing and pattern recognition theories and methods, combining traditional methods with deep learning, the method of character recognition in natural scenes is studied. It has wide application prospects in detonator code recognition, instrument reading recognition/remote meter reading, license plate recognition, container number recognition, natural scene text recognition, etc.
